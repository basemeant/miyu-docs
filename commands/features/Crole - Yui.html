Crole

Manage user-specific roles.

Before You Start:

  • Make sure one of Yui's roles has the Administrator permission.
  • Make sure the role you plan to assign is already created. Yui cannot create roles.

Usage:

Command
Description
y!crole set <member> <role>
Assigns the given role to the given member.
y!crole clear <member>
Clears the custom role from the given member.
y!crole list
Lists all custom roles in the server.

Related Commands

Rcolor
Used to recolor one's custom role.
Rname
Used to rename one's custom role.
