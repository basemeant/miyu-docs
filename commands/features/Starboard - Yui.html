Starboard

Manage a user maintained pin channel.

Usage:

Command
Description
y!starboard
Shows the current status of the starboard.
y!starboard bind
Binds the starboard to the current channel.
y!starboard disable
Disables the starboard.
y!starboard emote <emote>
Changes the starboard emote to the given emote.
y!starboard color <hex code>
Sets the starboard embed color to the given color.
y!starboard color random
Sets the starboard embed color to be random every time.
y!starboard threshold <number (1-30)>
Sets the starboard reaction threshold to the given number.
