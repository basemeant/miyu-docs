Tags

Manage Yui's tag feature.

Usage:

Command
Description
y!tags add <name> <message>
Creates a tag with the given name and message.
y!tags list
Lists all tags in the server.
y!tags list {member}
Lists all tags made by the given member.
y!tags info <name>
Shows info on the given tag.
y!tags edit <name> <message>
Changes the message of the given tag to the given string.
y!tags remove <name>
Deletes the given tag.
y!tags enable
Enables tags in the server.
y!tags disable
Disables tags in the server.
y!tags lock
Restricts tag creation/removal/editing to members with the manage guild permission.
y!tags unlock
Lifts tag creation/removal/editing restriction.

Example

Setup Command
Trigger Command
Outcome
y!tags add test {{0}}, this is a test tag.
y!t test John
John, this is a test tag.
