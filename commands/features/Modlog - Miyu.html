Modlog

Manage Miyu's modlog, which can log events that aren't recorded by discord's audit logs, such as edited/deleted messages.

Usage:

Command
Description
y!modlog
Shows the current status of the modlog.
y!modlog bind
Binds the modlog to the current channel.
y!modlog unbind
Unbinds the modlog.
y!modlog add <event>
Adds the given event to the modlog.
y!modlog remove <event>
Removes the given event from the modlog.
y!modlog set
Opens a prompt to add multiple events to the modlog at once.
y!modlog list
Shows all available events.
y!modlog enabled
Shows all currently enabled events.
