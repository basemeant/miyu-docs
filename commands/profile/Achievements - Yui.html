Achievements

View information about achievements.

Usage:

Command
Description
y!achievements
Shows your recent and close achievements.
y!achievements unlocked {page}
Shows the given page of your unlocked achievements (defaults to page 1).
y!achievements list {page}
Shows the given page of the achievements list (defaults to page 1).
y!achievements view <id>
Shows information about the given achievement.
