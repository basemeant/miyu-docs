Tickets

Manage Yui tickets.

Usage:

Command
Description
y!tickets
Shows information about tickets in the server.
y!tickets create
Opens a new ticket.
y!tickets create <topic>
Opens a new ticket with the given topic.
y!tickets create <member(s)>
Opens a new ticket and invites the given member(s).
y!tickets create <topic> <member(s)>
Opens a new ticket with the given topic and invites the given member(s).
y!tickets close <ticket channel(s)>
Closes the tagged ticket channel(s) and removes all participants.
y!tickets open <ticket channel(s)>
Opens the tagged ticket channel(s) and reinvites all participants.
y!tickets delete <ticket channel(s)>
Deletes the tagged ticket(s) and ticket channel(s).
y!tickets view <ticket channel>
Shows information about the tagged ticket channel's ticket.
