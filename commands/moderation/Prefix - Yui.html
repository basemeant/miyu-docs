Prefix

Changes Yui's command prefix.

Usage:

Command
Description
y!prefix <prefix>
Yui will only respond to commands that start with the specified prefix, Yui or @Yui#2222.
y!prefix none
Yui will only respond to commands that start with @Yui#2222.
y!prefix default
Yui will only respond to commands that start with y!, y., Yui or @Yui#2222.
Prefixes are not case sensitive and Yui will ignore any spaces or new lines between the prefix and the command.
