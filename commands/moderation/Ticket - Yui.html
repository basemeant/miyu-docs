Ticket

Manage a ticket from within a ticket channel.

Usage:

Command
Description
y!ticket
Shows information about the current ticket.
y!ticket view
Shows information about the current ticket.
y!ticket close
Marks the ticket as closed and removes all participants.
y!ticket open
Marks the ticket as open and reinvites all participants.
y!ticket delete
Deletes the ticket and ticket channel.
y!ticket topic <topic>
Changes the ticket topic to the given topic.
