Warnings

Manage warnings issued with Yui's warn command.

Usage:

Viewing Warnings

Command
Description
y!warnings
Lists all warnings issued in the server.
y!warnings list <member>
Lists all warnings issued to the given member.
y!warnings view <warning id>
Shows information on the given warning.

Managing Warnings

Command
Description
y!warnings remove <warning id>
Removes the given warning.
y!warnings clear <member>
Removes all warnings issued to the given member.
y!warnings clear
Clears all warnings in the server.
