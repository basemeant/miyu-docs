Blacklist

Prevents the mentioned member from using Yui in the server.

Usage:

Command
Description
y!blacklist
Shows the blacklist.
y!blacklist <member>
Blacklists the given member.

Related Commands

Whitelist
Used to remove a member from the blacklist.
Perms
Used to set up advanced permissions in a server.
