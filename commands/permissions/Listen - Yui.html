Listen

Allows Yui to respond to commands in the mentioned channel, if the channel was restricted by the ignore command (defaults to current channel).

Usage:

y!listen {channel}

Related Commands

Ignore
Used to prevent Yui from responding to commands in a channel.
Botchannel
Used to set a bot channel restriction in a server.
Freechannel
Used to remove the bot channel restriction from a server.
