Sub

Subscribe to self-assignable roles.

Usage:

Command
Description
y!sub
Shows a list of self-assignable roles in the server.
y!sub <name>
Subscribes to the given role.

Related Commands

Unsub
Used to unsubscribe from self-assignable roles.
SAS
Used to set up self-assignable roles in a server.
