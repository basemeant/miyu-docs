Stodo

Manage a server todo list with Miyu.

Usage:

Command
Description
y!stodo
Shows the server todo list.
y!stodo add <string>
Adds the given string to the server todo list.
y!stodo remove <string>
Removes the given string from the server todo list.
y!stodo clear
Clears the server todo list.
