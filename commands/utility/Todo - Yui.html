Todo

Manage a todo list with Yui.

Usage:

Command
Description
y!todo
Shows the todo list.
y!todo add <string>
Adds the given string to the todo list.
y!todo remove <string>
Removes the given string from the todo list.
y!todo clear
Clears the todo list.
