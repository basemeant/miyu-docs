Remindme

Set reminders with Yui.

Usage:

Command
Description
y!remindme <reminder> in <{xx days} {xx hours} {xx minutes}>
Sets the given reminder for the given amount of time in the future.
y!remindme list
Shows a list of upcoming reminders
y!remindme remove <id>
Removes the reminder with the given ID.
y!remindme clear
Clears all reminders.
