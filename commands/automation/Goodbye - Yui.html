Goodbye

Manage Yui's goodbye messages.

Usage:

Command
Description
y!goodbye bind
Binds goodbye messages to the current channel.
y!goodbye message <message>
Changes the contents of the goodbye message to the given string.
y!goodbye unbind
Turns off goodbye messages in the server.

Placeholders

You may add placeholders to the goodbye message which will be replaced when the goodbye message is posted.
Placeholder
Replaced By
{username}
The nickname of the user that left.
{server}
The name of the server.
{membercount}
The server's current member count.
{time}
The amount of time the member was in the server.
