Welcome

Manage Miyu's welcome messages.

Usage:

Command
Description
y!welcome bind
Binds goodbye messages to the current channel (Replaces welcome DMs).
y!welcome message <message>
Changes the contents of the goodbye message to the given string.
y!welcome unbind
Turns off goodbye messages in the server. Also applies to welcome DMs.
y!welcome dm
Welcome messages will be sent in a DM instead of a server channel (Replaces server welcome messages).

Placeholders

You may add placeholders to the welcome message which will be replaced when the welcome message is posted.
Placeholder
Replaced By
{mention}
The tag of the user that joined.
{username}
The nickname of the user that joined.
{server}
The name of the server.
{membercount}
The server's current member count.
