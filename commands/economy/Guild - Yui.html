Guild

Larger guilds can hold more workers. (Feature is currently in early development.)

Usage:

Command
Description
y!guild
Shows information about the guild.
y!guild upgrade
Upgrades the guild, allowing for more workers to be hired.
The guild funds feature has not been implemented yet.
